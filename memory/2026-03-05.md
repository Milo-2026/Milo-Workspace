- 00:04 EST: Found root cause of Apollo.io 403 error. According to Apollo.io documentation (https://docs.apollo.io/docs/create-api-key):
  1. API access depends on your plan - free plans may not have API access
  2. When creating a key, you must toggle "Set as master key" to select all endpoints
  3. The api_search endpoint requires a master key, not just any API key
  4. Key quote: "Some endpoints, like Get a List of Users, are only accessible with a master key."
  
  Solution: User needs to go to https://app.apollo.io/#/settings/integrations, connect Apollo API, create a new key with "Set as master key" toggled ON, then update .env with the new key.
