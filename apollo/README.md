# Apollo.io Lead Generation Setup

## Getting Started

1. Sign up at https://www.apollo.io
2. Get your API key from https://app.apollo.io/settings/api
3. Add to .env: `APOLLO_API_KEY=your_key_here`

## Usage

```bash
python3 apollo_search.py --title "Founder" --size "1-50" --location "United States" --limit 100
```
