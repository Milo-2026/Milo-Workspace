# Grocery List App MVP

A family-first grocery list app built with React Native + Expo.

## Quick Start

```bash
cd /Users/alfredoalvarez/.openclaw/workspace/PROJECTS/grocery-list-mvp
npm install
npx expo start
```

## Project Structure

```
grocery-list-mvp/
├── src/
│   ├── App.tsx          # Main app with UI
│   └── data/
│       └── store.ts     # Data models & logic
├── package.json
└── tsconfig.json
```

## Features Implemented (MVP)

- ✅ Create grocery lists
- ✅ Add items with auto-categorization
- ✅ Check off items
- ✅ Auto-sort by aisle (Produce, Dairy, Meat, etc.)
- ✅ Progress tracking
- ✅ Quick add modal
- ✅ Family-ready UI

## To Run Demo

Open `src/App.tsx` in an Expo-compatible editor or run:
```
npx expo start --web
```
