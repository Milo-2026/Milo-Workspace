# Life Directory - PARA System

This folder contains the long-term knowledge graph for the AI assistant.

## Structure

- **projects/** - Active projects with goals, progress, deadlines
- **areas/** - Ongoing areas of responsibility (finances, health, etc.)
- **resources/** - Reference materials, links, notes
- **archives/** - Completed projects, old references

## How It Works

1. During conversations, the AI records important facts here
2. Summary files provide quick lookups
3. Nightly consolidation extracts key info from daily notes
