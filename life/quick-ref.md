# Memory System Quick Reference

## 3-Layer Memory

### Layer 1: Knowledge Graph
- Location: `~/life/`
- PARA system: Projects, Areas, Resources, Archives
- Stores durable facts

### Layer 2: Daily Notes  
- Location: `~/memory/YYYY-MM-DD.md`
- Raw logs during conversations
- Extract to Layer 1 during consolidation

### Layer 3: Tacit Knowledge
- Location: `~/life/tacit-knowledge.md`
- Alfredo's preferences
- Hard rules: NO political, NO controversial
- Product context

## During Conversations

1. Note important things → daily notes
2. Remember preferences → tacit-knowledge
3. Update project status → life/projects/

## Session Startup

Always read:
1. SOUL.md
2. USER.md  
3. life/tacit-knowledge.md
4. memory/YYYY-MM-DD.md
5. MEMORY.md (main session only)
